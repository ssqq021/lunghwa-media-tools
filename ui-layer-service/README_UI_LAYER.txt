本地 UI 拆图服务

首次安装
1. 确保 C 盘至少有 65GB 可用空间。
2. 双击 install_ui_layer.bat，等待依赖安装完成。
3. 双击 start_ui_layer.bat，并保持窗口开启。
4. 打开网站里的“UI拆图”，点击“连接本地服务”。
5. 点击“准备本地模型”。首次会下载约 53.8GB 模型文件。

以后使用
1. 双击 start_ui_layer.bat。
2. 打开网站并进入“UI拆图”。

说明
- 服务只监听 127.0.0.1:7862，图片不会上传到外部服务器。
- 模型文件保存在本目录的 .model-cache 中，不会提交到网站。
- RTX 3060 12GB 使用 4 位量化与内存卸载。
- 忠实拆图使用模型生成的 Alpha 区域，并保留原图可见像素。
- 智能补全直接使用模型生成的透明 RGBA 图层。
- 最终由网页把可见图层切成多个透明 PNG，去除重复素材后打包 ZIP。
